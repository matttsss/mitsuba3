Dragon scene :

Dragon and a metal ball on a table rendered using an emitter sampling path-tracer
and an environment map for the background.

Credits :

    Dragon mesh :
        - Original 3D scan from the Stanford 3D Scanning Repository

        - Imported into Blender by user TrayClaine on BlendSwap website
            (https://www.blendswap.com/profile/866140)

        Captured by Bernhard Vogl at Wells, UK
